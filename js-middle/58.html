∴ Array 생성자 배열객체 구조

  • (*중요*) 객체 안에 여러 개의 배열을 만드는 방법
  • 결과적으로 2차원 배열구조를 가짐
  • (*중요*) 객체 접근시, car[0]와 같이 배열단위로 접근하거나, car[0][0]같이 배열안의 원소단위로 접근
        

    var car = new Array(
        [ 'Sonata', 'blue', 100 ],
        [ 'Jeep', 'red', 70 ],
        [ 'Passt', 'white', 150 ]
    )

    car[0][1]
    car[2][2]
    car[0]
