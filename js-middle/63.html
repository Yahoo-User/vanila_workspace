∴ 타이머 관련 메소드

  • 자바스크립트에서 타이머 설정은 2가지 메소드로 구현가능
  • Windows 객체의
      (1) setTimeout()
      (2) setInterval()
    메소드 사용
  • (*중요*) setTimeout() : 지정시간경과 후, 한번지정동작 수행
  • (*중요*) setInterval() : 동일지정시간 마다, 지정동작 반복 수행
  • 위 메소드들은, 특정시간 이후 함수 호출할 경우, 많이 사용하는 방법
  • 1000 밀리초 = 1초, 시간설정: 밀리초 단위로 작성
  1. setTimeout( function(), milliseconds )
  2. setInterval( function, milliseconds )
