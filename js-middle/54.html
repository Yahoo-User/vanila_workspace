∴ 자바스크립트 객체

  • (*중요*) 구분: (1) 사용자정의객체, (2) 내장객체
  • (1) 사용자정의객체(User-defined Object): 사용자가 직접 객체의 속성과 메소드를 정의하여 사용하는 객체
  •   예: Car(), House(), Hotel()
  • (2) 내장객체: 자바스크립트 언어 자체에서 정의하여, 사용자에게 제공하는 객체
  •   예: Object(), Array(), Date()
  • 자바스크립트는 웹브라우저에서 동작하는 프로그램
  • 웹브라우저는 웹브라우저 창과 이 창이 현재 보여주고 있는 웹 문서를 표현하기 위한 많은 내장객체를 가지고 있음
  • (*중요*) 내장객체의 종류(3가지):
    1. BOM(Browser Object Model, 브라우저 객체모델): 웹브라우저의 각종 요소를 객체로 표현:
      • navigator
      • screen
      • location
      • frames
      • history
      • XMLHttpRequest
    2. DOM(Document Object Model, 문서객체모델): 웹 문서의 각종 요소를 객체로 표현:
      • document
    3. GJO(Global Javascript Model, 전역자바스크립트객체): 자바스크립트 전체에서 사용가능한 내장객체:
      • Object
      • Array
      • Function
