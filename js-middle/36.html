ψ (2) 배열 객체(Array Object)로 생성하기

  • 자바스크립트에서 배열은 객체로 취급
  • (*중요*) 따라서, Array 생성자를 이용하여 배열객체 생성가능
  • (*중요*) 생성자를 사용하여 배열을 생성할 때에는 new 연산자 사용
  • 배열 객체 생성방법:
    var 배열명 = new Array(원소1, 원소2, ... )
  • 아래는 city 배열객체를 생성하고, 초기값을 할당한 예:
    var city = new Array('Seoul', 'Pusan', 'Incheon')
  • (*중요*) 배열 객체 역시, 배열선언과 초기값 할당을 따로 수행 가능:

        
    var city = new Array()      // 배열객체 생성(원소들 없이, 비어있는 배열)

    city[0] = 'Seoul'           // 배열에 개별 원소 할당
    city[1] = 'Pusan'           // 배열에 개별 원소 할당
    city[2] = 'Incheon'         // 배열에 개별 원소 할당
  • (*중요*) 배열객체로 배열을 생성할 경우, 배열에 공백 원소(데이터)를 사용불가!!!
  • (*중요*) 만일, 배열에 공백 데이터(원소)가 포함되어 있으면, 아래의 오류발생:
    Uncaught SyntaxError: Unexpected token
  • 배열객체의 생성 확인방법(3가지):
    1. typeof city : 타입확인연산자 typeof 사용, 결과: object
    2. Array.isArray(city) : 배열객체의 메소드인 isArray() 사용, 결과: true
    3. city instanceof Array : Array 생성자의 인스턴스인지 확인, 결과: true
