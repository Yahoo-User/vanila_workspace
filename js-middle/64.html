∴ 문서객체모델(DOM: Document Object Model)

  • (*중요*) 웹문서를 메모리로 읽어들여 트리(tree) 구조로 변환
  • (*중요*) 웹문서의 각종 요소는, 요소노드(element node)로 변환
  • (*중요*) 웹문서의 텍스트는, 텍스트 노드(text node)로 변환
  • 이렇게 변환된 트리구조를 이용하여, 자바스크립트로 웹문서 조작가능
  • (*중요*) 문서객체모델(DOM)의 트리구조를 순회(traverse)하면서, 웹문서의 구조, 내용, 스타일 등을 변경

1. 문서객체 생성

  • 웹문서 안에 문서객체를 포함시키면, 동적인 웹문서를 만들 수 있음:
  •   Step.1: 문서객체는 요소노드와 텍스트노드를 생성한 후, 요소노드에 텍스트노드를 연결
  •   Step.2: 요소노드를 body 객체에 추가하여 만듦

   문서객체(DOM) 생성 관련 메소드

  1. createElement() : 요소노드 생성
  2. createTextNode() : 텍스트노드 생성
  3. appendChild() : 요소노드를 body 객체에 추가

2. 문서객체의 속성값 설정

  문서객체의 속성값 설정 방법 2가지:

  1. 객체변수(참조변수) 사용 : 참조변수.속성 = 속성값
  2. 속성설정 메소드 사용 : setAttribute(속성, 값)

3. 문서객체의 스타일 설정

  문서객체의 스타일 설정은 아래와 같이 스타일 속성을 변경:

  document.getElemenetById(id).style.속성명 = 속성값
