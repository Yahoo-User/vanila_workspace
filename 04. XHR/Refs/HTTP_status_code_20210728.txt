* HTTP Status Code 의 의미:
  
 범위: 100 ~ 599 까지 정의되어있음 
 ---------------------------------------
  Status Code              의미
 ---------------------------------------
   100번 대(1XX) -> 현재 처리중
   200번 대(2XX) -> 정상처리 완료
   300번 대(3XX) -> Redirect / Cache 응답
   400번 대(4XX) -> Client 에서 잘못된 요청
   500번 대(5XX) -> Server 에서 오류가 발생
 ---------------------------------------
