• √ 변수의 자료형은 초기값에 의해 자동으로 결정
  • √ 변수에 저장된 값이 변경되면, 자료형도 자동으로 변경됨

  • √ 변수선언 시, var 키워드 생략가능
  • √ 같은 변수명을 재선언해도 해당 변수에 저장된 값이 없어지지 않음

  • √ 변수는 사용위치에 따라, "유효범위(scope)"를 가짐
  • √ 변수는 유효범위에 따라, (1) 전역변수 (2) 지역변수가 있음
  • √ 전역변수(Global variable): 자바스크립트 코드의 어느 위치에서든 선언하여, 코드의 전 영역에서 사용가능한 변수
  • √ 지역변수(Local variable): 함수블록에서 선언하여, 해당 함수블록 범위 내에서만 유효하게 사용가능한 변수
  • √ 함수 외부에서 선언한 변수가 전역변수, 함수내에서 선언한 변수를 지역변수로 구분 (***)
  • √ 전역변수는 var 키워드를 생략하고 선언할 수 있음
  • √ 함수블록 내부에서 var 키워드를 생략하고 선언한 변수는 자동으로 전역변수로 선언됨 (***)
