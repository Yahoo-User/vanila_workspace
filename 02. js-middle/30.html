∴ 반환값 출력

    

    function 함수명(매개변수1, 매개변수2, ...) {    // 함수선언
        실행문장들

        return 반환값
    } // 함수명


    // 함수호출결과 return 문이 반환한 값을 변수에 저장
    result = 함수명(전달인자1, 전달인자2, ...)      // 함수호출
  • 함수 수행 결과값을 반환할 때에는 return 키워드 사용
  • (*중요*) 만일 반환할 값이 없으면, return 문 생략가능
  • 반환값을 저장하기 위한 변수는 미리 선언해야 함
  • 함수호출이 끝나면, 자동으로 함수호출한 시점 다음으로 돌아가 계속 수행됨
  • 변수없이 함수호출 반환값을 출력하려면, 출력문에서 바로 함수호출하면 됨
  • (*중요*) 자바스크립트에서 함수는 변수에 할당하여 사용가능
  • 함수를 먼저 선언한 후, 함수명을 변수에 저장하는 방식
  • (*중요*) 여러 변수명으로, 하나의 함수를 참조(사용) 가능
