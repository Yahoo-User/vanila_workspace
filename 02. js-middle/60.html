∴ Date 객체의 메소드 목록

(1) getter 메소드

  1. getFullYear : 년도 반환
  2. getMonth : 0 ~ 11 월 반환
  3. getDate : 1 ~ 31 일 반환
  4. getDay : 0 ~ 6 요일 반환 (0: 일요일, 1: 월요일, ...)
  5. getHours : 0 ~ 23 시간 반환
  6. getMinutes : 0 ~ 59 분 반환
  7. getSecond : 0 ~ 59초 반환
  8. getMilliseconds: 0 ~ 999 밀리초 반환

(2) setter 메소드

  1. setFullYear : 1 ~ 31 날짜 설정
  2. setMonth : 0 ~ 11 월 설정
  3. setDate : 1 ~ 31 날짜 설정
  4. setDay : 0 ~ 6 요일 설정 (0: 일요일, 1: 월요일, ... )
  5. setHours : 0 ~ 23 시간 설정 (시간, 분, 초, 밀리초)
  6. setMinutes : 0 ~ 59 분 설정
  7. setSecond : 0 ~ 59 초 설정
  8. setMillisecond : 0 ~ 999 밀리초 설정
