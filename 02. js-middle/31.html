∴ 전달인자(arguments)와 매개변수(parameters)

    

    function 함수명(매개변수1, 매개변수2, ...) {    // 함수선언
        실행문장들

        return 반환값   // optional
    } // 함수명


    result = 함수명(전달인자1, 전달인자2, ...)      // 함수호출
  1. 전달인자와 매개변수는 개수도 같고, 데이터 타입도 같음
  2. 개수는 개발자가 작성
  3. (*중요*) 매개변수의 데이터 타입은, 전달인자의 데이터 타입으로 자동결정되므로, 따로 설정안함
  4. (*중요*) 함수명이 같고, 매개변수가 다른 함수가 여러개 있다면, 맨 마지막에 선언된 함수가 호출됨 (**함수 오버로딩 지원안함**)
  5. (*중요*) 전달인자의 개수가 0 이거나, 매개변수보다 개수가 적으면, NaN(Not-a-Number) 출력
  6. (*중요*) 전달인자의 개수가 매개변수보다 개수가 많으면, 첫 전달인자부터 적용된 후, 나머지 전달인자 값은 **undefined**로 처리
  • (*중요*) NaN(Not-a-Number) : 연산과정에서 잘못된 입력을 받았음을 나타내는 기호
  • (*중요*) 숫자를 반환하지 못할 때 출력
  • (*중요*) 자바스크립트에서 NaN은 전역변수임
  • NaN 여부를 판단가능한 함수 제공:
    isNaN(NaN) ⇒ true
    isNaN(0) ⇒ false
    isNaN('string') ⇒ true
    isNaN('0') ⇒ false
