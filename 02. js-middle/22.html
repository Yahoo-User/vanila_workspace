∴ For 문

    

    for(초기식; 조건식; 증감식) {

        실행문장들

    } // for

  • 초기식: counter 변수선언. 단 한번만 실행(*중요*)
  • 조건식: 얼마나 반복할지 결정. 조건식이 참 인동안 반복
  • 증감식: counter 변수의 값을 증가 또는 감소 시킴

∴ For 문의 변칙적 사용

  1. 초기식을 for 문 이전에 먼저 선언했다면, for 문에서 생략가능
  2. 예제코드: 
        
var num = 1

for( ; num <= 5; num++) {
    console.log('num =', num);
} // for
  3. 초기식은 여러개 선언가능:
  4. 예제코드: 
        
var num
var st = 'ABCDEF'
var ct


for(num = 1, ct = st.length; num < ct; num++) {
    console.log('num =', num, ', ct=', ct)
} // for
  5. for 문의 block 내에, 증감식 문장을 포함하면, for 문 자체에서 증감식을 생략가능
  6. 예제코드: 
        
var num
var st = 'ABCDEF'
var ct = st.length


for(  ; num < ct;  ) {
    console.log('num=', num)

    num++
} // for
  7. 초기식/조건식/증감식을 모두 생략하면, 무한루프 수행
  8. 예제코드: 
        
var num
var st = 'ABCDEF'
var ct = st.length

for( ; ; ) {

    if(num < ct) {
        console.log('num =', num)

        num++
        continue
    } // if

    break
} // for
