∴ 배열 메소드: sort

  • (*중요*) 배열의 원소를 정렬
  • (*중요*) 정렬의 결과: 원본배열도 정렬되고, 새로운 정렬된 배열도 반환
  • 영문자는 알파벳 순서로 정렬
  • 한글은 가나다 순으로 정렬
  • (*중요*) 영문자는, 대문자를 우선 정렬한 후, 소문자를 정렬
  • (*중요*) 따라서, 대소문자가 섞여잇는 경우, toLowerCase() 메소드를 사용하여,
    영문자를 모두 소문자로 변경한 후, 정렬해야 함
  • (*중요*) 숫자는 기본적으로 최상위 자리부터 오름차순으로 정렬
  • (*중요*) 숫자는 자릿수가 다른 수치가 저장되어있으면, 원하는 결과가 나오지 않을 수 있음:
    예: [ 13, 5, 28 ] → sort → [ 5, 13, 28 ]로 정렬되지 않고, [ 13, 28, 5 ]로 정렬됨

    이러한 문제는 아래와 같이 sort() 메소드 매개변수에, 별도의 두 숫자의 비교함수를 만들어 제공해서 해결해야 됨 

    
    // 오른차순 정렬을 원할 경우
    sort( function(a, b) {
        return a - b
    })

    // 내림차순 정렬을 원할 경우
    sort( function(a, b) {
        return b - a
    })
