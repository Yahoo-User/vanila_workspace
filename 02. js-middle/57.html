∴ 배열객체 (원소가 객체인 배열)

  • 표현방법 2가지:
    (1) 객체를 배열의 원소로 저장
    (2) 객체를 Array 생성자를 이용하여, 2차원 배열구조로 저장

(1) 객체를 배열의 원소로 저장

  • 객체를 배열의 원소로 저장하는 방법
  • (*중요*) 각 객체는 속성명과 값으로 이루어진 중괄호 단위로 작성
  • (*중요*) 객체에 접근할 때에는, 배열의 인덱스와 함께 속성명을 적음
        

    // 배열객체 리터럴 선언
    var car = [
        { name: 'Sonata', color: 'blue', speed: 100 },
        { name: 'Jeep', color: 'red', speed: 70 },
        { name: 'Passt', color: 'white', speed: 150 }
    ] 


    // 배열객체의 원소 및 원소의 멤버참조
    car[0].color        // 첫번째 원소 객체의 색상 참조(사용)
    car[2].speed        // 세번째 원소 객체의 속도 참조(사용)
    car[0]              // 첫번째 원소 객체 참조(사용)
