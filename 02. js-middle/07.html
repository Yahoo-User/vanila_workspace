A. 변수명 명명규칙 (*중요*)

  • var 키워드를 사용하여 선언
  • 특수문자를 제외한, 모든 영문자로 시작
  • 자바스크립트 언어의 예약어(Reserved word)를 변수명으로 사용불가
  • 숫자는 변수명의 첫문자로 사용불가
  • 문자, 밑줄(_), 달러기호($)로 시작가능
  • 대소문자 구별
  • 한글 변수명도 가능

B. 변수명 표기법

  1. 낙타 표기법 ⇒ userNumber, getLastName, setLastName
  2. 파스칼 표기법 ⇒ UserNumber, GetLastName, SetLastName
  3. 헝가리안 표기법 ⇒ numAge, strFirstName, strLastName

C. 변수 사용방법 (*중요*)

  • 우선 변수명을 선언하고(변수선언), 초기값 저장(변수정의)
  • 변수에 값을 저장할 때에는 대입연산자(=) 사용
  • 변수정의시, = (대입연산자)를 중심으로,
    좌변: 항상 변수명이 옴, 우변: 변수 / 표현식 / 리터럴이 올 수 있음
  •     
        var x                           // 변수 x 선언 
    
        var y = 10                      // 변수 y 선언 및 초기값 할당 ⇒ 변수정의 
    
        var x = y                       // 변수 y 값을, 변수 x에 저장 
    
        var a, b, c                     // 변수 a, b, c 선언 (변수정의가 아님!!!) 
    
        var a = 10, b = 20, c = 30      // 변수 a, b, c 선언 및 각각 초기값 할당 (3개 변수정의) 
    
        var a = b = c = 10              // 변수 a, b, c 선언 및 모두 동일한 10을 초기값으로 할당 (3개 변수정의) 
    
        var name = "홍길동", age = 25    // 변수 name, age 선언 및 각각 다른 초기값 할당 
    
        var total = a + b + c           // 변수 a, b, c 의 값을 모두 더한 결과를 변수 total 에 초기값으로 할당 (total 변수정의)
