∴ 연관배열(Association Array)

  • (*중요*) 배열의 인덱스를 "문자열"로 설정
  • (*중요*) 설정한 "인덱스(문자열)"에 주어진 값을 매핑
  • (*중요*) 즉, 0부터 시작하는 인덱스를 사용하는 대신, 특정 키워드(key_n)로 지정된 인덱스 사용
  • 연관배열의 생성방법:
    arr = { key_1: value1, key_2: value2, key_3: value3, ...., key_n: valueN }
  • (*중요*) 배열의 각 원소에 숫자 인덱스 대신에, 키워드로 설정된 인덱스 이용하여 접근
  • (*중요*) 원소의 참조(사용)방법:
    1. arr.keyN : keyN 키워드를 인덱스로 사용. 해당 값 참조
    2. arr[keyN] : 위와 동일
